Introducing Sentience.

Sentience is a Interactive Novel (Think Marvel comics but with animation and interaction) covering a few different stories im working on. I wont say too much about the stories at the moment, but I will talk about how I want it to function.

The game will cover content from three different story lines. You'll find a short story as you delve into each chapter. Under each short story you'll find a detailed scene with interactable objects. Each object will give you further insite to the story and world around it. Some objects will even be interactable. You might be able to start a mini game, or change the scene completely to expose the games secrets.

How and Why I'm doing it.

Sentience is basicaly a practice/passion project for me. I've had this idea for quite some time but havent been able to dedicate enough time to work on it. Well I've now challenged myself to build this game using only (mostly) vanilla languages with my main focus on Javascript and CSS. This gave me the excuse to work on this project for practice rather than just for fun. Using only vanilla languages means no Jquery, Angular functionality (although I am using it for ui-view/ui.router and other general things), SASS (other than nesting) and many other languages/libraries.

Technologies used so far:

Angular 1 | Vanilla JS | HTML | CSS | SASS | Gulp/Babel | Electron | Node.js
